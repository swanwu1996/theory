CTL的推导以及计算

含温度的缺陷形成能定义

\[ F^f(\alpha, q, T; \mu_e) = \underbrace{\left[F(D^q, T) - F(\text{host}, T)\right] - \sum_i n_i \mu_i(T)}_{\text{把原子从库里拿走、放进缺陷}} + q\left[ \underbrace{\mu_e(T)}_{\text{给/拿电子}} + \underbrace{\varepsilon_{\text{VBM}}(\text{host}, T)}_{\text{把 }\mu_e\text{ 锚到 VBM}(T)} \right] + \underbrace{E_{\text{corr}}(q; T)}_{\text{像电荷-势校正}} - k_B T \ln g_q \]

转变能级 \(\varepsilon_\alpha(q/q';T)\) 是“让两个电荷态的形成自由能相等的 \(\mu_e\)” ，即

\[ F^f(\alpha, q, T; \mu_e) = F^f(\alpha, q', T; \mu_e). \]

把上式两边相减解出 \(\mu_e\)：

\[ \mu_e = \frac{F^f(\alpha, q', T; 0) - F^f(\alpha, q, T; 0)}{q - q'}. \]

这里 \(\mu_e\) 是设 VBM 为 0 的“参考形成自由能”

\[ F^f(\alpha, q, T; 0) \equiv \left[F(D^q, T) - F(\text{host}, T)\right] - \sum_i n_i \mu_i(T) + q\varepsilon_{\text{VBM}}(\text{host}, T) + E_{\text{corr}}(q; T) - k_B T \ln g_q \]

即“把电子部分的 \(q\mu_e\) 项去掉”的形成自由能。于是得到相对 VBM(T) 的 CTL：

\[ \varepsilon_\alpha(q/q';T) \quad (\text{w.r.t. VBM}(T)) = \frac{F^f(\alpha, q', T; 0) - F^f(\alpha, q, T; 0)}{q - q'}. \]

上式给出的 \(\varepsilon_\alpha(q/q';T)\) 是以 host 的 VBM(T) 为能量零点的转变能级；若希望与 0 K 静态 DFT 结果对齐，需要再减去 host 的 \(\Delta \varepsilon_{\text{VBM}}(T)\)，得到相对于 VBM(0 K) 的 CTL。

CTL 计算路线（通用 q / q'）

\[ \varepsilon_\alpha(q/q';T) = \frac{\left[F^f(\alpha, q', T) - F^f(\alpha, q, T)\right]}{q - q'} - \left[\Delta\varepsilon_{\text{VBM}}(\text{host}, T)\right] \]

（含 \(T\) 的缺陷形成自由能差 / 每单位电荷） （参照能校正：VBM 随 \(T\) 的位移；基准是 VBM(0 K)）

展开公式

我们记

\[ \Delta F^f_{q-q'}(T) = F^f(\alpha, q', T) - F^f(\alpha, q, T) \]

\[ \Delta F^f_{q-q'}(T) = \underbrace{E_{\text{tot}}(D^{q'}) - E_{\text{tot}}(D^q)}_{\text{静态能差}} + \underbrace{\left[E_{\text{corr}}(q') - E_{\text{corr}}(q)\right]}_{\text{带电修正}} + \underbrace{\left[\Delta V_{\text{align}}(q') - \Delta V_{\text{align}}(q)\right]}_{\text{势对齐（含常量）}} + \underbrace{\left[\Delta F^q_{\text{vib}}(T) - \Delta F^q_{\text{vib}}(T)\right]}_{\text{振动自由能差}} - k_B T \ln \frac{g_{q'}}{g_q} \]

一般 \(q \rightarrow q'\) 情形下的显式结论公式

综合上面的推导，对一般的 \(q \rightarrow q'\) 情形，可以把有限温度下、相对于 VBM(0 K) 的转变能级写成一条非常直接可用的公式：

\[ \varepsilon_\alpha(q/q'; T) = \frac{ \Big[ \underbrace{\Delta F_{\text{FEP}}^{q \to q'}(T)}_{\text{算出的 FEP}} + \underbrace{\Delta E^{\text{FNV}}_{\text{corr}}(T)}_{\text{FNV + 势对齐差}} - k_B T \ln \frac{g_{q'}}{g_q} \Big] }{q - q'} - \Delta \varepsilon_{\text{VBM}}(\text{host}, T). \]

其中：

  • \(\Delta F_{\text{FEP}}^{q \to q'}(T)\)：由 MD/FEP 得到的缺陷在 \(q'\) 和 \(q\) 两个电荷态之间的有限温度自由能差（已经包含静态能 + 振动等一切在采样势能面上的贡献）。
  • \(\Delta E^{\text{FNV}}_{\text{corr}}(T)\)：两电荷态之间的 FNV 修正 + 势对齐差，即 \(E_{\text{corr}}^{\text{FNV}}(q';T) - E_{\text{corr}}^{\text{FNV}}(q;T)\)。
  • \(- k_B T \ln (g_{q'}/g_q)\)：来自简并度的熵贡献，如果 \(g_{q'} = g_q\) 可以忽略。
  • \(\Delta \varepsilon_{\text{VBM}}(\text{host}, T)\)：host 体系的 VBM 随温度的带边位移，用来把“相对于 VBM(T)”的 CTL 转换到“相对于 VBM(0 K)”的能量零点。

对于最常见的 \(0/+1\)、\(+1/+2\) 等情形，只需把相应的 \(q, q'\) 以及 \(g_q, g_{q'}\) 代入上式即可。
